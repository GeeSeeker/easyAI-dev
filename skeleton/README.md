# 我的项目

> 由 [easyAI](https://github.com/GeeSeeker/easyAI-dev) 框架驱动。

## 快速开始

1. 在 Antigravity IDE 中打开此项目
2. 输入 `/pm` 启动项目经理
3. 向 PM 描述你的需求，开始开发！

## 项目结构

```
.
├── .agents/        ← AI 角色层（规则、工作流、Skills）
├── .trellis/       ← 数据持久层（配置、规范、任务、日志）
├── .docs/          ← 用户文档空间
└── ...             ← 你的项目代码
```

## 框架升级

```bash
npx @geeseeker/easyai-dev update .
```
