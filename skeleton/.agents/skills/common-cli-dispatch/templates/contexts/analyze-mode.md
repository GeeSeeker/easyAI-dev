# 外部 CLI 上下文：分析模式 (Analyze Mode)

> 本模板由 `common-cli-dispatch` Skill 组装 Prompt 时加载。
> **场景**：PM 委派 CLI 进行原始需求的可行性分析或方案初步探讨。
> **目的**：获取外部模型的独立思考，不带主观预想。

---

## 模式守卫 (Guardrails)

当前会话目标是对给定的需求进行原始可行性与初步架构分析。
为了确保你的视角独立客观，主控 AI 已主动屏蔽了现有系统内部的分析结论与草案。请你秉持独立思考原则。

## 分析指示 (Instructions)

请基于上述输入要素进行专业分析。重点关注：

1. **技术可行性**：是否有违背客观规律或存在明显缺陷的部分？
2. **边缘情况**：有没有遗漏的关键边界用例？
3. **架构隐患**：长期演进是否存在潜在风险？
4. **决策参考**：给出建议以辅助主控 AI 进行后续的技术决策。
